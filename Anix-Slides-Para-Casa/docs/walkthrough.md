# Anix Slides - Solução Completa para Murais Digitais

O aplicativo agora é uma ferramenta completa para murais digitais em escolas e empresas.

## Novas Funcionalidades para TVs Smart

### 1. Modo Loop Automático
Perfeito para TVs de 60 polegadas. 
- Defina o tempo de transição (ex: 5 segundos).
- Ative o botão **Loop**.
- Ao clicar em **Apresentar**, o sistema percorrerá todos os slides infinitamente.

### 2. Gravação de Vídeo (Exportar)
Transforme sua apresentação em um arquivo de vídeo para usar onde quiser.
- Clique em **"Gravar Vídeo"**.
- O navegador solicitará que você selecione a aba para compartilhar.
- O sistema iniciará a apresentação automaticamente enquanto grava.
- Ao parar a partilha de ecrã, o arquivo `.webm` (compatível com a maioria das TVs modernas) será baixado automaticamente.

### 3. Submenu de Formas
Organização aprimorada com presets de Retângulo, Círculo e Quadrado para facilitar o design.

## Resumo das Ferramentas
- **Texto**: Com opção de "Tarja" (fundo) para legibilidade.
- **Imagem**: Upload local ou link da web.
- **Formas**: Arrastáveis e redimensionáveis.
- **Persistência**: Seus dados ficam salvos localmente no navegador.

## Como Usar na TV
1. Abra o link do app no navegador da TV.
2. Configure o Loop e a duração.
3. Clique em Apresentar.
4. O mural ficará rodando sozinho.
